How does CSS interact with HTML?

CSS (Cascading Style Sheets) can be thought of as another language like HTML.
It's used to define the styling of HTML elements

There are 3 ways we can write CSS:

1. Written in its own .css file
Cleanest solution 

index.css

.css-class {
  color: red;
  font-size: 18px;
}

2. Embedded in the style attribute of HTML elements
A bit messier, but not too bad 

<p style="color: red; font-size: 18px;">
  Test paragraph
</p>

3. Defined in the HTML file itself in a <style> HTML element
Kinda messy imo 

index.html

<html>
  <head>
    ...
    <style>
      .css-class {
        color: red;
        font-size: 18px;
      }
    </style>
  </head>

  <body>
    <p class="css-class">
      Test paragraph
    </p>
  </body>
</html>
