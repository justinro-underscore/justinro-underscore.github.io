Morse Code Project

Morse Code Project

SUMMARY

The Morse Code Project is a fun application that allows computers to communicate through the use of morse code projected through speakers at superaudible levels. The project, written in Java, has a user interface that is implemented using the JavaFX Framework. It utilizes high-level Digital Signal Processing libraries to parse audio data into patterns, and those patterns into text. The project is unfinished, but will be completed in the upcoming months.

SKILLS
  • Java
  • JavaFX
  • TarsosDSP Library
  • Audio I/O
  • UI Design
